## 🏨 Revenue Insights Dashboard — Hospitality Domain
**Tool:** Power BI &nbsp;|&nbsp; **Domain:** Hospitality &nbsp;|&nbsp; **Timeline:** Aug 2025 – Sep 2025

An interactive Power BI dashboard built to help hotel management track revenue performance, occupancy trends, and booking behaviour across multiple properties and cities.

## 🎯 Business Problem

Hotel chains managing multiple properties struggle to monitor KPIs like occupancy, ADR, and RevPAR across locations in real time. This dashboard provides a single view to identify underperforming properties and revenue gaps.

---

## 💡 Key Insights

- **Total Revenue:** ₹1.71 Billion across all properties
- **Overall Occupancy:** 57.87% with Weekend occupancy (62.64%) outperforming Weekday (55.99%)
- **Top Booking Platform:** LogTrip and Journey had highest Realisation % (~70.5%)
- **Business vs Luxury split:** Nearly 50-50 revenue share (50.24% Business, 49.76% Luxury)
- **ADR:** ₹12,696 average across all properties with RevPAR at ₹7,336

## 📌 Features

- 🔍 **Filter by City and Room Type** — dynamic slicers for drill-down
- 📈 **Trends by Key Matrix** — weekly RevPAR, ADR, and Occupancy % trend lines
- 🏨 **Property-Level Table** — bookings, cancellations, DSRN, DBRN, DURN, ratings per hotel
- 🍩 **Revenue by Category** — Business vs Luxury donut chart
- 📊 **Realisation % & ADR by Booking Platform** — platform performance comparison
